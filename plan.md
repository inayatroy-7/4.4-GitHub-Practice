# Plan Document

## Project Plan

### Goal
Practice using GitHub for a class assignment.

### Needs
- A GitHub account
- A public repository
- Knowledge of branches, commits, and pull requests

### Steps
1. Create a public repository
2. Add a README file
3. Create a dev branch
4. Make commits to the dev branch
5. Merge dev into main via a pull request

## Repo Maintenance Plan

### Goal
Keep the repo organized and up to date.

### Needs
- Clear commit messages
- Organized branches
- Closed issues when resolved

### Steps
1. Use issues to track tasks
2. Make commits with clear messages
3. Use branches for new changes
4. Merge completed work into main
